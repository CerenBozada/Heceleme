# Heceleme


Bu projede Türkçe dilindeki heceleme kuralları Duygun Altınok'un "Towards Turkish ASR: Anatomy of a rule-based Turkish g2p" adl makalesindeki heceleme kurallarından yararlanılarak bir algoritma oluşturulmuştur. Bu sayede Trük dilindeki sözcüklerin doğru hecelemesini saptayan bir program geliştirilmiştir.
